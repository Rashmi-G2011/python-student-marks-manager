# Student Marks Manager

A simple menu-driven Python application to manage student records and marks. This project is beginner-friendly and demonstrates the use of functions, dictionaries, loops, and conditional statements.

## Features

- Add a new student with marks
- View all student records
- Search for a student by name
- Update a student's marks
- Delete a student record
- Exit the program

## Technologies Used

- Python 3

## Concepts Used

- Functions
- Dictionary
- While Loop
- If-elif-else Statements
- User Input

## How to Run

1. Make sure Python 3 is installed.
2. Save the program as `student_marks_manager.py`.
3. Open a terminal or command prompt.
4. Run the program using:

```bash
python student_marks_manager.py
```

## Sample Menu

```
===== Student Marks Manager =====
1. Add Student
2. View Students
3. Search Student
4. Update Marks
5. Delete Student
6. Exit
```
